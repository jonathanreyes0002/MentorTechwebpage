## Hosting (Free options)

### Option 1 — GitHub Pages (Recommended)
1. Create a repo named `mentortech-website` on GitHub
2. Upload all files maintaining folder structure
3. Go to repo **Settings → Pages → Source: main branch / root**
4. Site live at: `https://yourusername.github.io/mentortech-website/`

### Option 2 — Netlify (Drag & drop, zero config)
1. Go to [netlify.com](https://netlify.com) → "Deploy manually"
2. Drag the `mentortech/` folder
3. Get a custom subdomain instantly: `mentortech.netlify.app`
4. Connect your domain `mentortech.org` for free

### Option 3 — Cloudflare Pages
1. Connect GitHub repo at [pages.cloudflare.com](https://pages.cloudflare.com)
2. No build command needed (static site)
3. Free custom domain + free CDN globally

### Option 4 — Vercel
1. Import GitHub repo at [vercel.com](https://vercel.com)
2. Zero configuration for static sites
3. Free tier generous, fast global CDN

## Recommendations
- **Easiest**: Netlify drag & drop  
- **Best for custom domain**: Cloudflare Pages (free SSL, global CDN)  
- **Best for version control workflow**: GitHub Pages
